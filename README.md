# Affordable and Portable Wireless Incubator for Premature Babies

## 📌 Project Description
This project aims to develop an affordable and portable wireless incubator for premature babies. It helps maintain the required temperature and provides a safe environment for newborns.

## 🎯 Objectives
- Low-cost incubator
- Portable design
- Wireless monitoring
- Temperature control

## 🛠 Technologies Used
- Arduino
- Sensors
- Embedded Systems
